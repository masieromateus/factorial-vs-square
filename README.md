# Factorial vs Square

A simple Python script to prove that $n! \le n^2$ is false for $n \ge 4$. 

## 💡 The Idea
I saw this math problem about informal proofs and wanted to see the exact point where the factorial function "breaks" the quadratic one. While it works for $n=1, 2, 3$, it fails at $n=4$.

My goal here is to:
1. Find the decimal "tipping point" between 4 and 5 using a more precise approach.
